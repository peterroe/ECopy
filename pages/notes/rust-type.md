## 标量类型（Scalar）

* 整型
* 浮点型
* 布尔型
* 字符型

### 整型

默认：`i32`

| 长度 | 有符号类型 | 无符号类型 |
| - | - | - |
| 8位  | i8        | u8     |
| 16位 | i16       | u16    |
| 32位 | i32       | u32    |
| 64位 | i64       | u64    |
| 128位| i128      | u128   |
| arch | isize     | usize  |

整型字面量：

| 数字字面量 | 实例 |
| - | - |
| 十进制 | 23_34|
| 十六进制 | 0xaf |
| 八进制 | 0o76 |
| 二进制 | 0b1011 |
| 字节（u8）| b'A' |

### 浮点数

`f32`和`f64`，默认为`f64`

### 布尔类型

`bool`

```rust
let isBoy: bool = false;
```

### 字符类型

`char`

```rust
let key: char = 'G';
```

## 复合类型

`rust`有两种基本复合类型：**元组**和**数组**

### 元组

```rust
let tuple: (i32, f64, bool) = (3, 3.14, false);

//支持解构
let (x, y, z) = tuple;

//支持访问下标
println!("first is {}", tuple.0); //=> 3
```

### 数组类型

相较于元组，数组独特之处在于数组内的元素只能是相同类型

```rust
let array = [1, 2, 3, 4, 5];

//声明类型
let array: [i32; 5] = [1, 2, 3, 4, 5];

//声明相同初始值元素的数组
let array = [3.14; 2];  //=> [3.14, 3.14]

//支持访问下标
println!("first element is {}", array[0]);
```

## 切片Slice类型

`slice`类型没有所有权，允许引用集合中一段连续的元素序列
